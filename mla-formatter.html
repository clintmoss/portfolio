MLA Formatter

HTML | CSS | Javascript

This was a personal project built with HTML, CSS and Javascript.

It is built with the purpose to avoid other major auto-formatters that are often organizing the information wrong. Instead of manually formatting each citation for different essays in school this program helps save time and mental space by not having to memorize all the different rules of formatting for MLA (which was the standard format for all classes).

The project focused on using vanilla javascript to target the DOM and change items as well as taking in information and manipulating it to output the correct text snippet.
