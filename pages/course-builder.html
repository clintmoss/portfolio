Course Builder

  • Problem: Instructional designers had to learn HTML and built poorly accessible online courses
  • Process: Install Grapes JS, customize interface, test, and release
  • Solution: Instead of coding web pages for online courses, they can use a custom course builder

This project was built using Grapes JS to help streamline the course creation process for instructional designers at Pima Community College. The instructional designers initially had to learn HTML and CSS to integrate custom content into the LMS. The course builder provides a code-free environment for instructors to drag and drop design elements to build their course.
