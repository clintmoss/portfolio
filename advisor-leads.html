Advisor Leads

React | Redux

Built for a Financial Advising firm as a marketplace to sell and buy leads for annuities.

The application uses React, MySQL, and Redux. This project was a for a client and was worked on with a senior developer. Together we created a database that held on to advisors (main users) and the leads. Connected to a separate website, the website would collect the leads and send them to the MySQL database. The database with the leads would then be parsed and shown only to the advisors that were in that same state.
